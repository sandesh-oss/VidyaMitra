View your app in AI Studio: https://ai.studio/apps/dabf550a-0893-4627-9da1-35dc4ada56ae

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
